# 🔢 Even Odd Checker

A simple web application that checks whether a number is Even or Odd.

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Features
- User can enter any number
- Click the "Check" button
- Instantly displays whether the number is Even or Odd
- Clean and simple user interface

## 💡 How It Works
The program takes input from the user and uses the modulus operator (`%`) in JavaScript to determine whether the number is divisible by 2.

If:
- Number % 2 === 0 → Even Number
- Otherwise → Odd Number
